# Order Download Folder

Programa para ordernar la carpeta de descargas teniendo en cuenta grupos de extensiones para los tipos de archivos.

## Grupos de extensiones

- Paquetes **(package)**
- Imagenes **(image)**
- Archivo Comprimidos **(compress)**
- Codigo **(code)**
- Video **(video)**
- Musica **(music)**
- Documentos **(doc)**
